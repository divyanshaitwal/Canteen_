# Food Delivery Website

A responsive food delivery website with multiple sections including hero section, menu, customer reviews, and more.

## Project Structure

The project consists of HTML component files, CSS stylesheets, and JavaScript functionality organized as follows:

### HTML Components
- 1herosec.html - Hero section
- 2work.html - How it works section
- 3featured.html - Featured items
- 5download.html - Download app section
- 6customer.html - Customer reviews
- 7updated.html - Updates section
- 8restaurants.html - Restaurant listings
- 10menu.html - Menu section
- 11about.html - About section
- 16message.html - Messaging section
- 17question.html - FAQ/Questions section
- 18footer.html - Footer section
- faq.html - FAQ page
- offers.html - Offers page
- privacy.html - Privacy policy page

### Stylesheets
All CSS files are located in the css/ directory and correspond to their respective HTML components.

### JavaScript
- 1herosec.js - Hero section functionality

### Images
Images are stored in the images/ directory.

## Usage

Open the HTML files in a web browser to view the different sections of the website. The components are designed to work together as part of a larger single-page or multi-page website.

## File Naming Convention

Files are numbered to indicate their order/position in the website layout. Files with "_old" suffix are previous versions kept for reference.

## Browser Compatibility

The website includes responsive.css for mobile and tablet compatibility.
